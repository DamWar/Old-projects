Function that implements Dijkstra's algorithm and shows the path.
Dijkstra's algorithm (or Dijkstra's Shortest Path First) is an algorithm for finding the shortest paths between nodes in a graph.(src https://en.wikipedia.org/wiki/Dijkstra%27s_algorithm)

Parameters
adj[]- vector of pairs of number of vertex and weight towards adjacent vertex (graph)
vertices- vector of vertices
findFor- index of vertex to search from
adjSize- size of vector adj
output- string containing shortest paths and their distances
